# Part A：方法论基础设施

## A.1 证据分级体系

五级分级体系，所有信息必须标注等级+来源+获取日期：

| 等级 | 定义 | 典型来源 |
|------|------|----------|
| **A — 官方/法定记录** | 政府机关、法院、立法机关正式文件 | 中选会公告、监察院财产申报、立法院公报、法院判决书 |
| **B — 权威媒体实名报道** | 具编辑审核流程的媒体机构产出 | 联合报/自由时报/中央社实名报道、具名专访、记者会逐字稿 |
| **C — 可追溯但单一来源** | 可指向具体出处但未经独立交叉验证 | 单一媒体独家、政论节目发言、智库报告（非同行评审）、维基百科 |
| **D — 弱来源/待核实** | 来源模糊、匿名或无法独立验证 | 匿名爆料、论坛传言、未署名社群截图 |
| **E — 分析推断** | 由分析者基于已有证据做出的逻辑推断 | 框架内所有分析结论 |

**E级框架标注体系**：

| 标注 | 含义 |
|------|------|
| [E-Hermann LTA] | 基于Hermann领导特质分析的推断 |
| [E-George OpCode] | 基于George操作码分析的推断 |
| [E-Winter Motive] | 基于Winter动机分析的推断 |
| [E-Barber Type] | 基于Barber类型学的推断 |
| [E-MBTI] | 基于MBTI认知功能分析的推断（须交叉验证） |
| [E-SNA] | 基于社会网络分析的推断 |
| [E-Composite] | 基于多框架综合的推断（最高可信度） |

**使用规则**：
- A/B级信息可直接纳入画像结论
- C级信息须标注"待交叉验证"并标注来源机构立场偏向
- D级信息仅可作为线索列入附录，不可纳入正文结论
- E级推断须明确列出所依据的事实编号、推理逻辑与所使用的学术框架

## A.2 时间分期框架

| 代号 | 时期 | 标志事件 | 国际环境 |
|------|------|---------|----------|
| P0 | 戒严时期（—1987） | 解严以前的政治社会化阶段 | 冷战格局、美中建交 |
| P1 | 民主化初期（1987—1996） | 解严、国会全面改选、总统直选 | 冷战结束、台海危机 |
| P2 | 首次政党轮替（1996—2008） | 李登辉后期至陈水扁两任 | 911后反恐、中国入WTO |
| P3 | 马英九时代（2008—2016） | 二次政党轮替、两岸大交流 | 金融危机、美国亚太再平衡 |
| P4 | 蔡英文时代（2016—2024） | 太阳花运动后效应、美中对抗升级 | 中美贸易战、COVID-19 |
| P5 | 赖清德时代（2024— ） | 少数政府、三党不过半 | 美中战略竞争深化 |

**使用规则**：
- 每条立场/表态须标注所属时期代号
- 立场转向分析须至少跨两个时期对照
- 可根据人物生涯增加自定义子时期（如P0-a蒋经国在世/P0-b后蒋经国时代）

## A.3 题目类型标记

| 标记 | 含义 | 执行要求 |
|------|------|---------|
| 📋 采集 | 事实信息采集 | 须提供可核验来源、证据等级 |
| 🔄 交叉验证 | 对已采集信息的独立核实 | 须引用不同于首次采集的来源 |
| 🔬 分析 | 基于事实的推断与评估 | 须列出所用框架、假设、替代解释，标注E级 |
| 📊 量化 | 需要数据统计或结构化产出 | 须给出数据来源、统计方法 |
| 🔗 关联 | 跨模块信息的交叉比对 | 须标注所关联的其他模块编号 |
| 🧠 建模 | 基于学术框架的行为建模 | 须明确标注所用框架、评估维度、推理链 |
| 🕸️ 网络 | 社会网络结构分析 | 须给出网络图、中心性指标、结构洞识别 |
| 🔮 推演 | 基于模型的前瞻推演 | 须列出假设条件、推演逻辑、不确定性评估 |

## A.4 信息缺口处理机制

每个模块在信息不足时必须使用以下格式显性标注：

```
【已获信息】：（实际采集到的内容与证据等级）
【信息缺口】：（未能获得的信息项）
【分析缺口】：（已有信息但尚未完成的分析维度）
【缺口原因】：
  □ 法定保密/未公开
  □ 来源存在但受限（付费墙/地理限制/语言障碍）
  □ 来源可能存在但未检索到（建议进一步检索方向：___）
  □ 信息本身不存在（该人物/该时期确无此类记录）
```

## A.5 台湾一手数据源标准（V7新增）

以下6类数据源为每份报告的必采项，获取后标注A级：

| 序号 | 数据源 | 官方网址 | 必采内容 | 采集方式 |
|------|--------|----------|----------|----------|
| 1 | 中央选举委员会(中选会) | db.cec.gov.tw | 历次选举各行政区得票数+得票率 | Perplexity/台湾IP直采 |
| 2 | 公民监督国会联盟(公督盟) | ccw.org.tw | 各会期评鉴结果+量化指标 | Perplexity/台湾IP直采 |
| 3 | 监察院阳光法案平台 | cy.gov.tw | 财产申报(不动产/存款/证券) | Perplexity/台湾IP直采 |
| 4 | 立法院公报系统 | lci.ly.gov.tw | 提案数/出席率/质询记录 | Perplexity/台湾IP直采 |
| 5 | 监察院政治献金查阅平台 | cy.gov.tw | 总收入/个人捐款/企业捐款 | Perplexity |
| 6 | 户政系统/官方简历 | — | 出生/学历/职历 | 多源交叉验证 |

若目标人物为非立法委员（如县市长、行政官员），则模块六/七对应调整为行政绩效指标。

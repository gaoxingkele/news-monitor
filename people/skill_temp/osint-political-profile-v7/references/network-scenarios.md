# Part D：网络分析与情景推演

## D.1 社会网络分析（SNA）

### 节点类型定义

| 节点类型 | 说明 | 示例 |
|---------|------|------|
| target | 分析目标人物 | 蒋万安 |
| political | 政治人物/政党/派系 | 朱立伦、民进党、黄复兴党部 |
| institutional | 政府/立法机构 | 台北市议会KMT团、AIT |
| economic | 企业/财团/公会 | 不动产公会、台智光 |
| academic | 学术/智库 | 宾大法学院 |
| media | 媒体/舆论场 | — |
| family | 家族成员 | 蒋孝严、蒋经国 |
| international | 国际行动者 | 东京都、美国国会 |

### 边属性定义

| 属性 | 说明 | 取值 |
|------|------|------|
| relation | 关系描述 | 自由文本（如"政治盟友/站台背书"） |
| weight | 关系强度 | 0.0-1.0（1.0=最强） |
| evidence | 证据等级 | A/B/C/D |
| direction | 方向性 | 有向边（影响力方向） |
| period | 关系活跃时期 | P0-P5 |

### networkx量化指标（V7必含）

使用 `data_collection/network_analysis.py` 计算以下指标，结果以表格形式嵌入报告§2.8：

| 指标 | 说明 | 报告呈现方式 |
|------|------|-------------|
| 度中心性(Degree Centrality) | 直接连接数量 | Top-10节点排名表 |
| 中介中心性(Betweenness Centrality) | 作为最短路径中间节点的频率 | Top-10节点排名表 |
| 接近中心性(Closeness Centrality) | 到达所有其他节点的平均距离 | Top-10节点排名表 |
| K-Core | 节点所属的最大k-核 | 核心-边缘结构说明 |
| 结构洞约束系数(Constraint) | Burt结构洞理论，低约束=占据结构洞 | 识别桥接角色 |
| 桥接边(Bridges) | 移除后增加连通分量的边 | 关键脆弱连接列表 |
| 网络密度(Density) | 实际边数/可能边数 | 整体连接紧密度 |

### 图谱构建流程

1. 从 `extracted/*.json` 中提取所有人物/组织关系
2. 补充手工编码的已知关系（党内/家族/国际）
3. 构建 networkx DiGraph
4. 计算量化指标
5. 输出到 `extracted/network_analysis.json`
6. 报告中以表格+文字分析呈现

### 网络演化分析

按P0-P5时期追踪网络变化：
- 哪些节点是新增的？（新盟友/新对手）
- 哪些边的权重发生了变化？（关系强化/疏远）
- 结构洞位置是否发生了转移？
- 桥接角色是否发生了更替？

## D.3 情景推演引擎

### 标准五情景模板（V7）

基于蒋万安实战经验，V7定义以下五个标准推演情景：

| 情景 | 名称 | 核心假设 | 关注变量 |
|------|------|---------|---------|
| A | 施政优异→全国竞选 | 市政绩效转化为全国声望 | 满意度、党内整合、对手弱化 |
| B | 两岸紧张升级 | 台海军事对峙或外交危机 | 家族符号被攻击、立场被迫清晰化 |
| C | 市政/治理危机 | 重大公共事故或施政失误 | 危机管理能力、媒体应对、责任归属 |
| D | 选举前置条件 | 九合一/立委选举结果影响 | 党内地位、盟友消长、民意趋势 |
| E | 台海高强度军事冲突 | RAND/CSIS级军事推演级别 | 安全立场、国际网络激活、政治存续 |

每个情景须包含：
1. **前提条件**：触发情景的具体事件假设
2. **行为预测**：基于五框架综合画像的行为推断
3. **概率评估**：标注估算方法（专家判断/历史基准/模型推算）
4. **影响评估**：对目标人物政治前途的影响方向和程度
5. **大陆应对建议**：针对该情景的接触/风险预案

### 自定义情景
